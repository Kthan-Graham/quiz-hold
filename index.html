Welcome to my capstone project!

Here you will find a quick little coding exerciese that will test your knowledge when it comes to programming.

It uses a combination of HTML, CSS, Javascript with a frame work of bootstrap. The reason why I chose this as my capstone is because I have gone through the classes but never implemented them together. This allows me to see how everything would interact.

Please keep in mind that this was originially suppose to be an app but sadly, I was not able to get Kivy working on my laptop so I had to make it into a dynamic website.

This is my first project trying to tie together HTML and Javascript which was intresting to say the least. I did also try to set up JEST to have some automated testing, even though I said I would just run through my self. I really wanted to focus on the ability to expand in the future.
